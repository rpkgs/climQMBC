## Quantile Mapping methods 

- Quantile Mapping (QM)
- Detrended Quantile Mapping (DQM)
- Quantile Delta Mapping (DQM)
- Unbiased Quantile Mapping (UQM)
- Scaled Distribution Mapping (SDM)
